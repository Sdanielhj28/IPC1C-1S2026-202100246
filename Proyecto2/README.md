## 📌 Descripción

El sistema permite gestionar un entorno académico completo, incluyendo la administración de usuarios, cursos, secciones, inscripciones, notas y reportes.

---

## ⚙️ Funcionalidades

- Gestión de usuarios:
  - Administrador
  - Instructor
  - Estudiante
- Creación de cursos y secciones
- Inscripción y desasignación de estudiantes
- Registro de notas
- Cálculo de promedios por sección
- Generación de reportes en PDF
- Carga masiva mediante archivos CSV
- Bitácora de eventos del sistema
- Monitoreo en tiempo real mediante hilos
